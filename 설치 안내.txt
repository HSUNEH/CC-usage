================================
  CC-usage 설치 안내
  Claude Code 사용량 모니터
================================

[ 1. 앱 설치 ]

  CC-usage.app을 Applications 폴더로 드래그하세요.


[ 2. 첫 실행 시 "손상되었습니다" 경고가 뜨는 경우 ]

  Apple 서명이 없는 앱이라 macOS가 차단합니다.
  터미널을 열고 아래 명령어를 실행하세요:

  xattr -cr /Applications/CC-usage.app

  (Applications가 아닌 다른 경로에 있다면 경로를 맞춰주세요)
  실행 후 앱을 다시 열면 정상 동작합니다.


[ 3. Claude Code 로그인 (최초 1회) ]

  앱이 사용량 데이터를 가져오려면 Claude Code에
  로그인되어 있어야 합니다.

  터미널에서:

  npm install -g @anthropic-ai/claude-code
  claude

  브라우저에서 OAuth 로그인이 진행됩니다.
  로그인 완료 후 앱을 새로고침하면 사용량이 표시됩니다.


[ 4. 사용법 ]

  - 앱이 30초마다 자동으로 사용량을 업데이트합니다
  - 터미널 CLI, 웹(claude.ai) 어디서 사용하든 반영됩니다
  - 우측 하단 "API" 뱃지가 보이면 정상 연결된 상태입니다
  - 메뉴바에 남은 시간과 사용률이 표시됩니다
